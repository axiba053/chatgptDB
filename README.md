# chatgptDB

[![Streamlit App](https://static.streamlit.io/badges/streamlit_badge_black_white.svg)](https://chatgptdb.streamlit.app)

## 功能
* ChatGPT正常对话
* ChatGPT知识库对话
* ChatGPT联网对话


## 安装说明

1. 安装依赖
```commandline
pip install -r requirements.txt
```

2. 配置.env文件
```
OPENAI_API_KEY= ''
QDRANT_COLLECTION_NAME=''
QDRANT_HOST=''
QDRANT_API_KEY=''
```

3. 运行
```commandline
streamlit run 1_🏠_主页.py
```
